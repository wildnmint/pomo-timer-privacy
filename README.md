# Pomo Timer Privacy Policy

Effective date: June 30, 2026

Pomo Timer is a local-only Chrome extension. It does not transmit, sell, or share user data with the developer or third parties.

## Data stored locally

Pomo Timer uses `chrome.storage.local` to save:

- The current task name and up to five recent task names
- Timer settings, selected mode, current timer state, and completion timestamp
- Theme and optional sound preferences
- Daily completed focus-session count and focus minutes

This information is used only to provide the extension's timer, task history, appearance, sound, and daily-stat features. Task names are entered by the user and may contain personal information, so users should avoid entering sensitive information they do not want stored in their Chrome profile.

## Data transmission and sharing

Pomo Timer has no backend, analytics, tracking, advertising, external API calls, remote scripts, or remote audio files. Stored data remains in Chrome's extension storage on the user's device and is not transmitted by Pomo Timer.

## Retention and deletion

Data remains in local extension storage until the user selects **Clear local data** inside Pomo Timer or uninstalls the extension. Clearing local data stops the current timer and removes saved tasks, settings, theme, sound preferences, and stats.

## Permissions

Pomo Timer uses Chrome permissions only for local storage, timer alarms, completion notifications, and optional locally generated completion sounds.

## Changes

This policy may be updated if Pomo Timer's data practices change. The effective date above identifies the current version.

## Contact

Privacy questions can be sent through the support contact listed on Pomo Timer's Chrome Web Store page.
